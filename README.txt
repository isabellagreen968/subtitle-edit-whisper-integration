üîß CUSTOMIZATION GUIDE
----------------------

1. CHANGING COLORS
   - Open style.css
   - Edit the :root section at the top (lines 1-10)
   - Change values for --color-primary, --color-secondary, etc.
   - Save and refresh browser

2. REPLACING GOOGLE FORM LINK
   - Open contact.html
   - Find the "Open Contact Form" button (around line 110)
   - Replace the href URL with your new Google Form link
   - Keep target="_blank" and rel="noopener noreferrer"

3. UPDATING GITHUB LINK
   - Find GitHub links in all HTML files (search for "github.com")
   - Replace: https://github.com/isabellagreen968/SubtitleEdit
   - With your repository URL

4. CHANGING DOWNLOAD LINK
   - Open download.html
   - Find the "Download Now" button (around line 100)
   - Replace: https://subtitleedit.net/download/
   - With your download page URL

5. UPDATING FOCUS KEYWORDS & LINKS
   - Open index.html
   - Find the "platform-info" section
   - Three keyword links are naturally placed in paragraphs:
     * "subtitle edit official website" ‚Üí https://subtitleedit.net/
     * "subtitle edit linux" ‚Üí https://subtitleedit.net/subtitle-edit-on-linux/
     * "subtitle edit online" ‚Üí https://subtitleedit.net/free-online-subtitle-tools-for-your-videos/
   - To change: Replace both the text and href attribute

6. EDITING TEXT CONTENT
   - Open any HTML file in a text editor
   - Find the section you want to edit
   - Change text between HTML tags
   - Keep HTML structure intact (don't remove tags)

7. CHANGING FONTS
   - Open style.css
   - Find the Google Fonts import at the top of HTML files
   - Replace font names in the URL
   - Update font-family declarations in CSS
   - Current fonts: Inter (body), Poppins (headings)

8. ADDING SOCIAL MEDIA LINKS
   - Open footer section in any HTML file
   - Find the "social-links" div
   - Copy existing social-link structure
   - Add new SVG icon and link
   - Update across all pages for consistency
